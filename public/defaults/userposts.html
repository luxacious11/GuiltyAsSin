While we aren't exactly a novel, we're still a place for writing and so we'd like to reward you for any writing activity you participate in on-site this month! This includes: character applications, full-length threads, comms threads, and overall word counts. There will be points assigned for the first three and goal posts available for points and word counts and you'll earn prizes for reaching each goal post. There will also be ranking prizes for the top three points earners.


I'll do my best to update the Tracking Spreadsheet at least once a week. Points Please note that a wanted ad is considered as a single character wanted ad or a general wanted ad with only listed roles. If you have a group wanted ad, you will get this value multiplied by the number of characters - assuming that each character as their own write up of at least a few lines. Additionally, one-shots placed in your dev board can count as a full length post!

  • Per Wanted: 3 point

Claim Code If you intend to participate, post this form once in this thread and then edit it to include all your characters, comm posts, full length posts, wanted ads, and word counts.   Posting Templates
lux Posted on: August 14 2021, 11:54 AM

Replies: 2
Views: 129 		need something fancier? We got you covered! This one has a lot more options, and most of them are related to inserting an extra piece of code, so we're going to go through this one step at a time in regards to what's available.

First Step Most importantly, everything after this needs to be within a wrapper tag. This wrapper tag only needs one single attribute: data-size. This was discussed above and, if you don't use it, it'll just default to the medium width.

CODE
<tag-wrap data-size="sizeValue">
other code here
</tag-wrap>

Sections First, we have sections. Sections are like this bit that you're reading right now. These use the data-bg and data-color options, but they also have one extra attribute available: data-extraPadding. This adds some extra blank space either to the top or the bottom depending on which value you place (and those are the values)! If you don't need it, just leave it out. In addition, you can include a second copy of this section within itself to create this look we have going here - where there's a different colored background inside of a main section.

CODE
<tag-section data-bg="backgroundColor" data-color="fontColor" data-extraPadding="paddingLocation">
content here
</tag-section>

Character/Quote Section This section provides the option for a quote, a character image, or both, similar to the character header in the template immediately above this one. The code below is for both; don't want the image? Remove the img tag. And the text? Just don't put any in if you don't want any. Otherwise, it uses the same data attributes as the tag-section.

CODE
<tag-quote data-bg="backgroundColor" data-color="fontColor" data-extraPadding="paddingLocation">
<img src="http://placehold.it/100" />
content here
</tag-quote>

Dividers What that paper divider? Easy enough! This does not use the color or background attributes - but you can decide which color is above and below the paper (data-preBG and data-postBG respectively). The values for these should match the background value for your section above the paper for data-preBG and the background value for your section below the paper for data-postBG.

CODE
<tag-paper data-preBG="preBGValue" data-postBG="postBGValue"><img src="https://files.jcink.net/uploads2/playedgod/2021/thinEdge.png"></tag-paper>

Images If you want an image, but not the one in the quote section, you're welcome to use the specific tag-image option. This gives you attributes for three size options, left/center/right alignment, and border color:

Size

  • Square: square
  • Tall: tall
  • Wide: wide
Alignment
  • Left: left
  • Right: right
  • Center: center
Border
  • None: square
  • Primary Color: main
  • Alternate Color: alt
  • Bold Accent: accent
  • Mid Accent: accent2
  • Faded Accent: accent3
CODE
<tag-image data-size="sizeValue" data-border="borderValue" data-align="alignValue">
<img src="http://placehold.it/600">
</tag-image>

Views: 67 		What about our propaganda posters? You can use those too! This template also does not use the data attributes. It does, however, allow you to set a ratio on the image based on a couple major photo ratios. I've listed them below for you, just place them where it says RATIOVALUE.
  • 1:1 — r1-1
  • 4:3 — r4-3
  • 16:9 — r16-9
  • 3:2 — r3-2
  • 8:5 — r8-5

[dohtml]<div class="chap-propaganda"><div class="chap-propInner">

<span class="ratio RATIOVALUE"><img src="http://placehold.it/400"></span>
